    API

    Diseñada para producción, no para el demo

    Autenticación robusta y semántica de reintentos segura desde la primera llamada. Sandbox con datos realistas antes de tocar dinero real.

    • Firma HMAC por peticiónCada request se firma con tu llave secreta; nadie opera sin ella, ni siquiera con el token.
    • Anti-replayTimestamp y nonce obligatorios: una petición interceptada no puede reutilizarse.
    • Idempotencia nativaRepite una llamada con la misma llave y el pago se ejecuta una sola vez. Sin duplicados por timeout.
    POST /v1/transferencias
    # Dispersión con idempotencia y firma HMAC
curl -X POST https://api.masmovil.lat/v1/transferencias \
  -H "Authorization: Bearer $TOKEN" \
  -H "X-Idempotency-Key: nomina-2026-07-15-0042" \
  -H "X-Signature: $(firma_hmac $BODY)" \
  -H "X-Timestamp: 1783468800" \
  -d '{
    "clabe_origen":  "646180111200004821",
    "clabe_destino": "012320001234567895",
    "monto":         "18450.00",
    "concepto":      "Nómina quincena 13"
  }'

    Seguridad y cumplimiento

    Construido para pasar la revisión de riesgo

    Controles diseñados con los criterios de un analista de riesgo en mente: trazabilidad total, autorización fuerte y capacidad de frenar la operación en segundos.

    PLD/AML con screening y scoring

    Verificación contra listas restringidas, scoring de riesgo por contraparte y alertas de operación inusual antes de liquidar.

    2FA obligatorio

    Segundo factor requerido para todos los usuarios y para cada acción sensible: dispersar, crear llaves o cambiar límites.

    Kill switch

    Un control de emergencia detiene toda salida de fondos al instante, por entidad o global, con bitácora de quién y cuándo.

    CEP de Banxico en cada operación

    Comprobante Electrónico de Pago disponible por API y panel para el 100% de las transferencias. Evidencia oficial, siempre.

    Para analistas de STP: con gusto compartimos nuestra matriz de controles, políticas PLD y arquitectura de seguridad en detalle. Escríbenos a direccion@masmovil.lat.
